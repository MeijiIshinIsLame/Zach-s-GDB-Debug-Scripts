# Zach's GDB Debug Scripts

GDB helper scripts to automate debugging steps and make my life easier.
Feel free to use or edit as you please.

## Usage

In `gdb`, run:

`source path/to/repository/importall.py`

### List of Commands

| Command   | Description                                                             |
|----------|-------------------------------------------------------------------------|
| `logsteps` | Logs the address, instruction, and registers for a specified number of steps to a file. |
